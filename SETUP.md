# 🎯 Setup OnMess - Instruções Iniciais

Você recebeu 7 arquivos. Aqui está o que fazer com eles:

## 📥 Arquivos Recebidos

1. **index.html** - Interface do app (abra isto primeiro!)
2. **app.js** - Lógica completa
3. **sw.js** - Service Worker (funciona offline)
4. **manifest.json** - Configuração para instalar como app
5. **README.md** - Documentação completa
6. **PUBLICAR.md** - Guia de publicação
7. **package.json** - Metadados do projeto

## ⚡ Começar Agora (3 passos)

### Opção A: Usar Localmente (SEM publicar)

1. **Crie uma pasta** chamada `onmess` no seu computador
2. **Coloque os 7 arquivos** dentro da pasta
3. **Abra `index.html`** no navegador (duplo clique ou arraste para o navegador)
4. **Pronto!** O app funciona completamente offline

> ✅ Melhor para: testes, uso privado, desenvolvimento

### Opção B: Publicar Online (COM um clique)

**Escolha um dos 3 mais fáceis:**

#### 1️⃣ GitHub Pages (GRÁTIS - Recomendado)
- Siga o guia em `PUBLICAR.md` (seção "GitHub Pages - 5 minutos")
- URL será: `https://seu-usuario-github.github.io/onmess/`

#### 2️⃣ Vercel (GRÁTIS - Mais rápido)
- Siga o guia em `PUBLICAR.md` (seção "Com Vercel")
- URL será: `https://onmess.vercel.app`

#### 3️⃣ Seu Servidor (PAGO - Controle total)
- Coloque os arquivos no seu hosting
- Ative HTTPS
- Siga o guia em `PUBLICAR.md` (seção "Seu Servidor Web")

> ✅ Escolha qualquer uma - todas funcionam igual

## 📚 O que o App Faz

### 5 Abas Principais

| Aba | O que faz |
|-----|-----------|
| 📅 **Agenda** | Cria eventos, gera contratos, assinatura digital |
| 🎤 **Artistas** | Perfil, press kit (fotos, rider, contratos) |
| 🎵 **Músicas** | Registra músicas, assinatura de colaboradores |
| 💰 **Finanças** | Receitas automáticas + despesas + prestação de contas |
| 🧰 **Fornec** | Base de fornecedores de som, hotel, transporte, etc |

### Bonus: Abas Adicionais

- 📇 **Contatos** - Base automática (contratantes + fornecedores)
- 👥 **Usuários** - Criar equipe com permissões diferentes
